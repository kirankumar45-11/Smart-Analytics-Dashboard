# Smart Analytics Dashboard

A responsive web-based analytics dashboard built using HTML, CSS, and JavaScript with interactive charts, dark mode, and dynamic UI.

## 🚀 Features
- Interactive charts using Chart.js
- Dark mode toggle
- Sidebar navigation
- Dynamic UI updates
- Responsive design

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript
- Chart.js

## 📊 Description
This project visualizes business data using charts and dashboards. It helps users understand data trends and make better decisions through an intuitive and modern interface.
